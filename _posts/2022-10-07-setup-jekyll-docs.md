## select theme
- go to `https://rubygems.org` and search for `jekyll-theme`
- add theme name to `Gemfile` 
- run `bundle install`
- go to `_config.yml` and update theme variable to your selected theme
- run `bundle exec jekyll serve` or `jekyll serve`
